# Features

The Business Variable panel provides an extensive set of features.

| Section                                          | Description                                       |
| ------------------------------------------------ | ------------------------------------------------- |
| [Sort by status](https://grafana.com/docs/plugins/volkovlabs-variable-panel/<PLUGINS_VERSION>/features/sort/)                        | Sort by any field from the data source            |
| [Key-values format in variables](https://grafana.com/docs/plugins/volkovlabs-variable-panel/<PLUGINS_VERSION>/features/key/)         | Use the key-values format in variables            |
| [Empty value in multi-value variables](https://grafana.com/docs/plugins/volkovlabs-variable-panel/<PLUGINS_VERSION>/features/empty/) | Empty value in multi-value variables              |
| [Persistent mode](https://grafana.com/docs/plugins/volkovlabs-variable-panel/<PLUGINS_VERSION>/features/persistent/)                 | Allows to keep the selected values if unavailable |
| [Redirect dashboards](https://grafana.com/docs/plugins/volkovlabs-variable-panel/<PLUGINS_VERSION>/features/redirect/)               | Open another dashboard from the Variable panel    |
| [Input text box](https://grafana.com/docs/plugins/volkovlabs-variable-panel/<PLUGINS_VERSION>/features/input/)                       | Input Text Box                                    |
